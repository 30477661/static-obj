本馆的总藏书数量为13万册。有5个书库分别为：基本书库、竞赛书库、教师研修室（工具书、样本书）和语文实验室2个（中外文学名著及传记）。基本书库、竞赛书库面向全校师生全开架借阅，教师研修室收藏有现刊220种、报纸30种及各类工具书和过刊杂志，供老师阅览不外借，语文实验室供高一、高二学生上阅读课，不外借。同时有学生阅览室2个分别为:学科阅览室、综合阅览室。三楼有学生电视台，四楼为名师工作室。 部门设置及职责： 采编室工作人员岗位职责: 　　 　　1. 根据中学图书馆的性质、任务，按照科学性、系统性、节约性的原则，力争以 最少的经费采购到最适合本馆读者需要的图书，而且复本量合理。 　　2. 凡订购的书籍到货后，负责与学校验收人员一起拆包、验收、登录财产帐目。 　　3. 类分图书准确、恰当。 　　4. 电脑录入目录快速、准确。 　　5. 认真做好工作记录，进行馆藏统计。 　　 基本书库工作人员工作岗位职责: 　　 　　1. 负责班图书干事的任命、管理及读者的培训工作。 　　2. 负责学生的借还书工作。 　　3. 负责新书的推介与宣传。 　　4. 负责新书的验收等。 　　5. 负责毕业班学生图书催还工作。 　　 　　 教职工阅览室工作人员职责: 　　 　　1. 根据本校教职工需要及经费条件，订购、采集有关报刊资料。 　　2. 整理加工报刊资料。 　　3. 负责本室报刊资料的保管，流通和阅览工作。 　　4. 做好对读者的宣传、推荐和辅导工作。定期进行读者阅读情况分析，有针对性 地加强阅读辅导。 　 5. 有针对性地对有关资料进行加工整理，主动做好文献编制工作，及时为学校教 育、教学、教研工作提供服务。 　　6. 负责本室的卫生清洁工作。 　　 　　 学生阅览室工作人员岗位职责: 　　 　　1. 根据本馆的性质、任务订购采集各种报刊。 　　2. 整理加工各种报刊。 　　3. 负责本室书报刊的管理和流通及阅览工作。 　　4. 按时足量开放阅览室、热情为学生服务，经常 巡查 ，及时整理、还架。 　　5. 负责各项读书活动的开展。 　　6. 教育读者爱护公物，包括图书及各种阅览设施。 　　7. 认真做好读者统计和借阅统计工作，定期进行本校学生阅读情况的分析，研究 学生的阅读心理和阅读倾向，有针对性的加强阅读指导。 　　8. 保持本室卫生清洁。 图书馆工作人员守则: 　　 　　1. 坚定正确的政治方向;热爱社会主义祖国，坚持四项基本原则，忠诚人民的教育 事业。 　　2. 热情主动的服务精神:树立"师生第一"的思想，经常调查研究，虚心听取意见， 不断改进服务态度，提高服务水平。 　　3. 认真负责的工作态度:认真履行岗位责任，积极完成本职工作，勇于探索，不断 提高工作效率和工作质量。 　　4. 自觉严格的纪律观念:遵守作息时间，不迟到，不早退，严格遵守学校的请假制 度，工作时间不会客，不聊天，不看小说，不做与工作无关的事情。 　　5. 团结友爱的同志关系:同志间要以诚相待，团结协作，要开展批评与自我批评。 　　6. 文明礼貌的道德风尚:尊重社会公德，讲究文明礼貌，建设良好作风，保持良好 馆容。
为了充分发挥我校图书馆的作用，使之更好地为广大师生的教学服务，现将图书馆外借处学生借书事宜及有关制度知知如下： 　　1. 读者凭借书证借书 　　借书证只限本人使用，借书时须出示借书证，做到人、证相符，以防假冒，因转借、代借而造成的一切后果均由原持证人负责。 　　2. 借书程序、数量及期限 　　要借书的同学先由本人在书库找到自己所要的图书，由本人到借台办理借书手续。学生看完后由本人来办理还书手续。 　　学生每人借书指标为3本，借书期限一个月。教师25本。 　　3. 借书超期罚款制度 　　借书逾期不还者，停止借书，同时予以罚款，罚款额每本每日0.1元，寒暑假不在逾期天数内。如书遗失，办理赔书手续前的超期天数亦按此条例罚款。 　　4. 图书遗失、污损、盗窃处理规定 　　所借图书如有遗失，限时以版本相同的图书赔偿，并交加工费三元，赔偿同种书确有困难者，则视图书情况按原价2—10倍罚款。 　　所借图书如有污损，则按污损程度处以程度不等的罚款，被污损的图书经罚款处理后，仍归图书馆。 　　凡对书刊资料撕页“开天窗”者，视情节轻重处以10—50元罚款，并写出检讨，情节严重者，报告学校按有关规定处理。 　　凡未办理借阅手续将书刊私自携出室外者，以盗窃书刊论处，除写出检讨外，按所窃图书价格10倍罚款，并交保卫部门按有关规定处理。 　　5. 学生毕业，须还清在图书馆所借图书，并交回借书证，未还清者，按丢失图书处以罚款，并扣留毕业证。 　　6. 本处实行电脑自动化管理，读者进入时不能妄动该处的工作电脑。 　　希同学们努力与本室工作人员配合！
